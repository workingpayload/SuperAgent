# React Ranger Skill

A specialized skill for creating, debugging, fixing, and improving React code with high efficiency and production-ready quality.

## Features
- Deep expertise in React: functional components, hooks, Context API, state management, routing, and performance optimization.
- Proficient with modern ecosystems: TypeScript, Vite, Next.js, and various UI libraries.
- Automatically generates/updates a `walkthrough.md` documenting all changes made.
- Ensures clean, maintainable, scalable architecture and runnable code.

## Installation

This skill can be installed globally for Gemini CLI by copying this directory to `~/.gemini/skills/react-ranger`.

## Usage

When using the CLI, simply ask to build, fix, debug, or improve React code. The skill will take over to provide expert-level React development while maintaining comprehensive documentation of its actions via `walkthrough.md`.
