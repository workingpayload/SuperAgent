# FastAPI Furious Skill

A specialized skill for creating, debugging, fixing, and improving FastAPI code with high efficiency and production-ready quality.

## Features
- Deep expertise in FastAPI: routing, async programming, dependency injection, Pydantic models, SQLAlchemy, and authentication.
- Automatically generates/updates a `walkthrough.md` documenting all changes made.
- Ensures clean, maintainable architecture and runnable code.

## Installation

This skill can be installed globally for Gemini CLI by copying this directory to `~/.gemini/skills/fastapi-furious`.

## Usage

When using the CLI, simply ask to build, fix, debug, or improve FastAPI code. The skill will take over to provide expert-level FastAPI development while maintaining comprehensive documentation of its actions via `walkthrough.md`.
